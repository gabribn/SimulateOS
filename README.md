# 🖥️ SimulateOS

Este é um projeto de simulação de conceitos de Sistema Operacional desenvolvido com Angular.

Há documentação sobre **fluxos de memória (com swap e sem swap)**, peculiaridades do simulador e **principais ações/funções** (NGXS, `BlocksState`, etc.):

## 🚀 Como Rodar a Aplicação

Siga os passos abaixo para configurar e executar a aplicação em seu ambiente local.

### 1. Pré-requisitos

Para rodar este projeto, você precisa ter as seguintes ferramentas instaladas em sua máquina:

* **Node.js:** Ambiente de execução necessário para o Angular e npm.
    * **Versão Recomendada:** **Node.js 14 LTS** ou **Node.js 16 LTS**.
* **npm:** Gerenciador de pacotes do Node.js (instalado automaticamente com o Node.js).
* **Angular CLI:** Interface de Linha de Comando do Angular, usada para executar os comandos de `start` e `build`.

#### Instalação Global do Angular CLI

Caso ainda não tenha o Angular CLI instalado globalmente, use o comando:

```bash
npm install -g @angular/cli
